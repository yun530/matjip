# 🗺️ 쩝쩝박사지도 서비스 기획서 v1.1

> 광고와 알 수 없는 리뷰어 대신, 내가 신뢰하는 지인의 실제 추천으로 맛집을 찾는 **폐쇄형 소셜 맛집 플랫폼**

---

## 1. 서비스 개요

### 1.1 한 줄 정의
지인 추천 기반의 폐쇄형 맛집 공유 플랫폼. "우리끼리만 아는 진짜 맛집과 거를 집을 투명하게 공유한다."

### 1.2 핵심 가치
- **신뢰성**: 모르는 사람의 별점이 아닌, 내가 아는 친구의 별점과 코멘트.
- **폐쇄성**: 상용화 계획 없음. 운영자와 친구들 중심의 끈끈한 네트워크.
- **투명성**: 광고 원천 차단. 솔직한 리뷰(개맛있는 집 vs 가지말아야 할 집) 지향.

---

## 2. 핵심 기능 및 정책 (Update)

### 2.1 핀(Pin) 색상 로직
지도 위에서 맛집의 '검증도'를 시각적으로 즉시 파악할 수 있도록 핀 색상을 차별화한다.
- **🔴 빨간색**: 5명 이상의 지인이 추천한 '검증된 맛집'
- **🟡 노란색**: 3명 이상의 지인이 추천한 '관심 맛집'
- **🔵 파란색**: 1명 이상이 추천한 '맛집'
- *핀 클릭 시 추천한 지인 명단 및 상세 리뷰 노출*

### 2.2 맛집 등록 규칙 (필수 항목)
단순 등록을 넘어 정보의 질을 높이기 위해 다음 항목을 필수값으로 지정한다.
- **검색**: 카카오 장소 검색 연동
- **별점 (텍스트 가이드)**:
    - 5점: "인생 맛집 (무조건 가야함)"
    - 4점: "자주 감 (믿고 먹음)"
    - 3점: "나쁘지 않음 (평타)"
    - 2점: "그저 그럼"
    - 1점: "절대 가지마 (비추)"
- **추천 메뉴**: "여기선 이걸 먹어라" 하는 핵심 메뉴 태그
- **코멘트**: 솔직한 한마디 (최대한 가감 없이)

### 2.3 네트워크 정책
- **팔로우 제한**: 쾌적한 UX와 긴밀한 정보 공유를 위해 **팔로잉 10명 이내 권장**.
- **피드 정렬**: 최신 등록 순서대로 정렬하여 친구들의 실시간 활동을 확인.
- **공개 범위**: '우리 사이에 비밀은 없다'. 모든 게시물은 팔로워에게 전체 공개.

---

## 3. 화면 구조 및 UI 전략

### 3.1 주요 탭
1. **🏠 홈 (최신 피드)**: 친구들의 최신 소식 타임라인.
2. **🗺 탐색 (지도)**: 내 주변 핀 지도. 추천 수에 따른 색상 차별화.
3. **➕ 등록 (추가)**: 장소 검색 및 필수 정보 입력.
4. **👤 프로필**: 내 추천 목록 및 팔로워 관리.

### 3.2 UX 원칙
- **직관적 신뢰**: 핀 색상만으로도 맛집의 등급을 알 수 있게.
- **솔직한 분위기**: 기성 서비스의 딱딱한 말투 대신, 친구에게 말하듯 편한 톤앤매너 지향.
- **카테고리 간소화**: 초기에는 불필요한 카테고리 필터를 제거하고 전량 노출하여 정보량을 확보.

---

## 4. 기술 스택 (확정)

- **Frontend**: Next.js 14 (App Router)
- **Styling**: Modern Vanilla CSS (프리미엄 & 클린 디자인)
- **Map**: Kakao Maps API (국내 최적화)
- **Backend/DB**: Supabase (인증, DB, 스토리지 통합 환경)
